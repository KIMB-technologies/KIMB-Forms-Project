Einträge mit diesem Gerät

Beim Ausfüllen einer Umfrage wird im Browser ein Code zum Löschen des Eintrags abgelegt.
Mit diesem Code können Einträge direkt mit dem Button "Meinen Eintrag löschen" auf der Seite der Umfrage gelöscht werden.

Einträge über die E-Mail-Adresse abfragen

Wurde ein Eintrag auf einem anderen Gerät vorgenommen oder taucht nicht mehr in der Liste oben auf, so kann hier eine Liste mit allen Einträgen für eine bestimmte E-Mail-Adresse angefragt werden.
Die Liste wird an die angegebene E-Mail-Adresse versendet und beinhaltet auch die Links zum Löschen der Einträge.
Achtung, es kann sein, dass Anfragen nicht für alle E-Mail-Adresse freigeschaltet sind.
%%CAPTCHA%%
Falls Einträge gefunden wurden, haben wir eine E-Mail verschickt.
